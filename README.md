# Password-Generator
🔐 Password Generator

👋 About
This is a simple password generator I built using Python.
It creates random passwords based on what the user wants — like length and whether to include uppercase letters, numbers, or special characters.
I made this project while learning Python, so it’s beginner-friendly and easy to understand.

✨ What it can do
-> Generate random passwords
-> Let you choose password length
-> Option to include:
  - Uppercase letters
  - Numbers
  - Special characters
- >Can generate multiple passwords in one run

🛠️ Built with
-> Python
->"random" module
->"string" module

▶️ How to use
1. Download or clone this repository
2. Open the folder in terminal / command prompt
3. Run the file:
python password.py
4. Enter your choices and get your password

💡 Example
=== Password Generator ===
Enter password length: 8
Include uppercase? (yes/no): yes
Include digits? (yes/no): yes
Include special characters? (yes/no): no
Generated Password: A8kLm2Qr
